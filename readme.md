# Project: Pig

## Description
The Pig game is a simple dice game where two players take turns rolling a single die. The objective of the game is to be the first player to reach a certain number of points, typically 100.

## How to Run
To run the Pig game, follow these steps:

1. Make sure you have Python installed on your computer.
2. Download the "pig.py" file from the project repository.
3. Open a terminal or command prompt and navigate to the directory where the "pig.py" file is located.
4. Run the following command: `python pig.py`.
5. Follow the prompts on the screen to play the game.

## Rules
1. The game starts with Player 1.
2. On each turn, a player can choose to roll a die or hold their current score.
3. If a player rolls a 1, their turn ends, and they don't score any points for that round.
4. If a player rolls any other number, that number is added to their round score.
5. Players can continue rolling as long as they want or until they choose to hold their current score.
6. If a player holds their score, their round score is added to their total score, and it becomes the other player's turn.
7. The first player to reach or exceed the target score (e.g., 100) wins the game.

# Project: Timed Math Challenge

## Description
The Timed Math Challenge is a simple game where players are presented with math problems and must solve them within a specified time limit. It's a great way to practice basic arithmetic operations.

## How to Run
To run the Timed Math Challenge, follow these steps:

1. Make sure you have Python installed on your computer.
2. Download the "timed_math_challenge.py" file from the project repository.
3. Open a terminal or command prompt and navigate to the directory where the "timed_math_challenge.py" file is located.
4. Run the following command: `python timed_math_challenge.py`.
5. Follow the prompts on the screen to start the game.

## Rules
1. The game starts by presenting a math problem to the player.
2. The player must enter their answer within a specified time limit (e.g., 10 seconds).
3. If the player enters the correct answer before time runs out, they earn a point.
4. If the player answers incorrectly or fails to answer within the time limit, they don't earn a point.
5. After each problem, the next problem is presented.
6. The game continues until the player decides to quit or reaches a predetermined number of problems.
7. The final score is displayed at the end of the game.

# Project: Madlibs Generator

## Description
The Madlibs Generator is a fun project that generates humorous stories by replacing certain words in a template with user-provided inputs. It allows users to create unique and entertaining stories each time they play.

## How to Run
To run the Madlibs Generator, follow these steps:

1. Make sure you have Python installed on your computer.
2. Download the "madlibs_generator.py" file from the project repository.
3. Open a terminal or command prompt and navigate to the directory where the "madlibs_generator.py" file is located.
4. Run the following command: `python madlibs_generator.py`.
5. Follow the prompts on the screen to generate your own madlibs story.

## Rules
1. The game starts by presenting a story template with blank spaces where words are missing.
2. The player is prompted to enter specific types of words (e.g., nouns, adjectives, verbs).
3. Once the player enters the words, the program replaces the blanks in the template with the provided words.
4. The completed story is then displayed, creating a humorous and unique narrative.
5. Players can play multiple rounds with different story templates by restarting the program.

Note: Make sure to have fun and let your creativity shine while playing the Madlibs Generator!
